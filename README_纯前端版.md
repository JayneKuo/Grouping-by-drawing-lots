# 球搭子网球赛事管理系统 - 纯前端版

## 🎯 专为非技术人员设计

- ✅ **纯前端应用** - 无需后端服务器
- ✅ **数据存储在浏览器** - 使用localStorage
- ✅ **支持导出/导入** - 可以保存数据文件
- ✅ **支持增删改查** - 所有操作都在浏览器中完成
- ✅ **直接部署Vercel** - 只部署前端，无需配置后端

## 🚀 3步快速部署

### 步骤1：推送到GitHub
将代码推送到GitHub仓库

### 步骤2：部署到Vercel
1. 访问 https://vercel.com/dashboard
2. 点击 **Add New Project**
3. 导入你的GitHub仓库
4. 点击 **Deploy**
5. 等待1-2分钟

### 步骤3：完成！
访问你的Vercel域名就可以使用了！

**就这么简单，无需配置任何后端服务！**

## 💡 功能说明

### ✅ 支持的功能
- ✅ 创建比赛
- ✅ 添加选手（单个或批量）
- ✅ 删除选手
- ✅ 录入比分
- ✅ 查看积分榜
- ✅ 导出/导入数据
- ✅ 多用户同步（通过导出/导入文件）

### 📦 数据存储
- 所有数据存储在浏览器的localStorage中
- 支持导出为JSON文件
- 支持导入JSON文件恢复数据
- 支持多用户通过文件共享数据

## ⚠️ 重要提示

1. **数据备份**：建议定期导出数据备份
2. **数据同步**：多用户通过导出/导入文件同步
3. **存储限制**：localStorage有5-10MB限制，足够小型比赛使用
