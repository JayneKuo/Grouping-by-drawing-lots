# 使用说明

## 快速开始

### 1. 打开应用
直接用浏览器打开 `index.html` 文件

### 2. 登录
- 点击"立即登录"或右上角"登录"按钮
- 使用测试账号：
  - **管理员**: `admin` / `admin123`
  - **用户1**: `user1` / `user123`
  - **用户2**: `user2` / `user123`

### 3. 创建比赛
1. 点击"创建新比赛"
2. 填写比赛信息：
   - **比赛名称**: 例如"我的球搭子年终互撕总决赛"
   - **赛制选择**: 短盘制/三盘两胜/五盘三胜
   - **计分方式**: 金球制/占先制
   - **分组方式**: 2组/4组/无分组
   - **报名人员**: 每行一个选手姓名
3. 点击"保存比赛"

### 4. 抽签分组
1. 进入比赛详情
2. 点击"开始抽签分组"
3. 系统自动均分，可手动调整

### 5. 录入比分
1. 在比赛列表中点击比赛
2. 找到对应场次，点击"录入"
3. 使用逐分录入系统：
   - 点击"得分"记录每一分
   - 点击"ACE"记录ACE球
   - 点击"OUT"、"发球失误"等记录特殊情况
   - 系统自动计算局分、盘分

### 6. 查看排名
- 积分榜自动更新
- 按多维度排序：积分 → 局数差 → 直胜 → ACE数 → 净胜分

### 7. 大屏展示
1. 打开 `display.html`
2. 通过URL参数选择场次：`display.html?matchId=xxx`
3. 或点击"切换场次"选择
4. 点击"全屏"进入全屏模式

## 功能说明

### 占先制 vs 金球制

**占先制（Advantage）**:
- 40-40后，需要领先2分才能获胜
- 40-40 → AD-40 → Game（获胜）
- 40-40 → 40-AD → 回到40-40

**金球制（No-Ad）**:
- 40-40后，下一分直接决定胜负
- 40-40 → 下一分 → Game（获胜）

### 抢七规则
- 触发条件：局数达到6-6（短盘制为3-3）
- 规则：先到7分且领先2分
- 发球：第1分由先发球方发，之后每2分换发球

### 平分出线排序
当积分相同时，按以下顺序排序：
1. **局数差** (gamesWon - gamesLost)
2. **直胜关系** (两人直接对战结果)
3. **ACE数差**
4. **净胜分**
5. **人工裁定**

## 快捷键（未来扩展）
- `1` - 选手1得分
- `2` - 选手2得分
- `A` - ACE
- `O` - OUT
- `F` - 发球失误
- `D` - 双误
- `L` - 重发
- `U` - 撤销

## 注意事项

1. **数据存储**: 所有数据保存在浏览器localStorage，清除浏览器数据会丢失
2. **多标签页**: 支持多标签页同步，修改会自动同步
3. **网络状态**: 断网时数据会保存到本地缓存，联网后自动同步
4. **浏览器兼容**: 建议使用Chrome、Edge等现代浏览器

## 故障排查

### 页面无法加载
1. 检查浏览器控制台（F12）的错误信息
2. 确认所有JS文件都在 `js/` 目录下
3. 检查文件路径是否正确

### 数据丢失
1. 检查localStorage是否被清除
2. 查看浏览器存储限制
3. 尝试清除缓存后重新登录

### 比分不同步
1. 检查网络连接
2. 刷新页面
3. 检查localStorage事件是否正常

## 技术支持

如有问题，请：
1. 打开浏览器开发者工具（F12）
2. 查看Console标签页的错误信息
3. 查看Network标签页的网络请求
4. 提供错误信息以便排查
