# 🎾 球搭子网球赛事管理系统 - 使用说明

## 🚀 启动方式

### 方式1：使用npm（推荐）

```bash
npm start
```

或者

```bash
npm run dev
```

这会自动：
- 启动本地服务器（端口8000）
- 自动在浏览器中打开应用

### 方式2：使用Python

```bash
python -m http.server 8000
```

然后手动访问：http://localhost:8000

### 方式3：双击启动脚本

- Windows: 双击 `启动.bat`
- PowerShell: 运行 `.\启动.ps1`

## 📋 功能说明

### 1. 登录系统
- 默认账号：`admin` / `admin123`
- 其他测试账号：`user1` / `user123`, `user2` / `user123`

### 2. 创建比赛
- 设置比赛名称
- 选择赛制（短盘制/三盘两胜/五盘三胜）
- 选择计分方式（金球制/占先制）
- 选择分组方式（2组/4组/无分组）
- 添加参赛选手

### 3. 分组抽签
- 点击"开始抽签分组"
- 系统自动随机分组

### 4. 录入比分
- 逐分录入系统
- 支持ACE、OUT、发球失误等操作
- 自动计算局分、盘分
- 支持抢七规则

### 5. 查看积分榜
- 自动计算积分
- 显示出线情况

## 💾 数据存储

- 所有数据保存在浏览器的 **localStorage** 中
- 刷新页面不会丢失数据
- 清除浏览器缓存会删除数据

## 🔧 技术栈

- HTML5 + CSS3 + JavaScript
- 模块化设计
- 响应式布局（支持手机和电脑）

## 📱 浏览器支持

- Chrome 60+
- Firefox 55+
- Edge 79+
- Safari 11+

## 🆘 常见问题

### Q: 页面显示空白？
A: 按F12打开控制台，查看错误信息。确保使用本地服务器运行。

### Q: 数据丢失了？
A: 检查是否清除了浏览器缓存。数据保存在localStorage中。

### Q: 无法启动服务器？
A: 确保已安装Node.js或Python。使用 `npm start` 最简单。
