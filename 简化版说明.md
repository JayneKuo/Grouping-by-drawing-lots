# 简化版使用说明（无需MySQL）

如果您的电脑没有安装MySQL，或者不想配置数据库，可以使用**纯前端版本**。

## 🚀 快速启动（纯前端）

### 方式1：使用Vite开发服务器
```bash
npm install
npm run dev
```

### 方式2：使用http-server
```bash
npm install -g http-server
http-server -p 8080
```

## 📝 纯前端版本特点

- ✅ 所有数据保存在浏览器的localStorage中
- ✅ 无需数据库配置
- ✅ 无需后端服务器
- ✅ 适合单机使用

## 🔄 切换到完整版（MySQL+后端）

如果以后想使用完整版本：

1. 安装MySQL: https://dev.mysql.com/downloads/mysql/
2. 运行 `自动配置数据库.bat`
3. 启动后端：`cd backend && npm start`
4. 启动前端：`npm run dev`

## 💡 建议

- **个人使用/测试**：使用纯前端版本
- **多人协作/生产环境**：使用MySQL+后端版本
